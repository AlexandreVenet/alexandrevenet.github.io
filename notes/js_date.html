JS : date

Rappel rapide à propos des dates en Javascript.

Déclarer un objet Date :


const currentDate = new Date();

Exemple de méthodes :


console.log(`Nous sommes le ${currentDate.getDate()}/${parseInt([currentDate.getMonth()]) + 1}/${currentDate.getFullYear()}. Il est ${currentDate.getHours()}:${currentDate.getMinutes()}`);

La méthode toLocaleDateString() permet traduction et mise en forme : 


const options = {
	weekday: 'long',
	year: 'numeric',
	month: 'long',
	day: 'numeric'
};
console.log(`Nous sommes le ${currentDate.toLocaleDateString('fr-FR',options)}.`);
console.log(`Heute ist ${currentDate.toLocaleDateString('de-DE', options)}.`);
console.log(`Today is ${currentDate.toLocaleDateString('en-GB', options)}.`);
console.log(`Oggi siamo ${currentDate.toLocaleDateString('it-IT', options)}.`);
